# GPT2 Python/C++ HF Implementation

This directory contains the Python and C++ implementation for integrating HoloQubed with Hugging Face GPT-2 models.

## Files
* `geometry_forge.py`: Transforms dense Flatland AI models into Complex Hilbert Phase Space for GPT-2.
* `holo_ext.cpp`: Native OpenCL C++ Extension for GPT-2 models.
* `holo_generate_ext.py`: Text generation engine using OpenCL extension for GPT-2.
* `holo_loader.py`: Ingress loader and CPU Query Planner.
* `holo_setup.py`: Standard setup script for building and installing the C++ PyTorch extension.
