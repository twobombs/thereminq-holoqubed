# Qwen3.5 Python/C++ HF Integration

This directory contains the Python and C++ implementation for integrating HoloQubed with Hugging Face Qwen3.5 models.

## Files
* `geometry_forge.py`: Transforms dense Flatland AI models into Complex Hilbert Phase Space for Qwen3.5.
* `holo_ext.cpp`: Native OpenCL C++ Extension for Qwen3.5 models.
* `holo_generate_ext.py`: Text generation engine using OpenCL extension for Qwen3.5.
* `holo_loader.py`: Ingress loader and CPU Query Planner.
* `holo_setup.py`: Standard setup script for building and installing the C++ PyTorch extension.
