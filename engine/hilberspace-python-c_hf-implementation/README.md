# Hilberspace Python/C++ HF Implementation

This directory contains the implementation for Hilberspace with Python and C++ integration for Hugging Face models.

## Files
* `geometry_forge.py`: Transforms dense Flatland AI models into Complex Hilbert Phase Space.
* `holo_ext.cpp`: Native OpenCL C++ Extension.
* `holo_generate_ext.py`: Text generation engine using OpenCL extension.
* `holo_loader.py`: Ingress loader and CPU Query Planner.
* `holo_setup.py`: Standard setup script for building and installing the C++ PyTorch extension.
